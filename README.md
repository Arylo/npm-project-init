## Usages / 使用方法

### New Project / 新建项目

#### Method 0 / 方法0(推荐)

```shell
npx arylo-init new <new_project_path>
```

#### Method 1 / 方法1

```shell
npm install arylo-init --global
arylo-init new <new_project_path>
```
### Update Project / 更新项目

#### Method 0 / 方法0(推荐)

```shell
npx arylo-init update <update_project_path>
```

#### Method 1 / 方法1

```shell
npm install arylo-init --global
arylo-init update <update_project_path>
```
